# Contributing

Спасибо за участие в проекте Tinkoff Invest!

## Быстрый старт

Рекомендуем использовать [poetry](https://pypi.org/project/poetry/)

```
pip install poetry
make install
```

### Запуск тестов

```
make test
```

### Запуск линтеров

```
make lint
```

### Запуск автоформатирования

```
make format
```
